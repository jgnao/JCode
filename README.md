# JCode

JCode é um editor de código desktop desenvolvido com React, Tauri e CodeMirror.

Em progresso..

## Recursos

- Explorador de arquivos
- Sistema de abas
- Salvamento de arquivos
- Interface moderna e leve

## Tecnologias

- React
- JavaScript
- Tauri
- Rust
- CodeMirror

## Executando o projeto

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
